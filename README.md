Cryptocurrencies such as Bitcoin and Ethereum are highly volatile digital assets whose prices fluctuate rapidly due to market demand, investor sentiment, global events, and speculative trading. Understanding this volatility is crucial for evaluating financial risk and making informed investment decisions.

This project focuses on analyzing historical cryptocurrency price data and applying statistical and mathematical models to measure market instability. The simulator calculates daily returns, percentage changes, and standard deviation to determine how much the price deviates from its average value over time. By visualizing this data through interactive graphs and simulations, the dashboard transforms complex numerical information into meaningful insights.

A key component of the project is risk assessment through probability-based modeling. Using Monte Carlo simulation techniques, the system generates multiple possible future price scenarios based on historical trends. This allows users to observe potential best-case, average-case, and worst-case outcomes, helping them understand uncertainty in financial markets.

The Crypto Volatility Simulator integrates concepts from mathematics, statistics, economics, and computer science. It demonstrates how real-world financial problems can be analyzed using programming tools and data visualization libraries. The project not only strengthens technical skills in data handling and dashboard development but also enhances financial literacy by explaining volatility, risk measurement, and forecasting in a practical and interactive manner.

Overall, this simulator serves as both an educational and analytical tool, showing how technology can simplify complex financial concepts while supporting data-driven decision-making in the digital economy.
